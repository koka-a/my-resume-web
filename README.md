# 个人响应式简历网页 (My Resume Web)

这是我的个人简历展示页面，旨在展示我的前端开发技能和 Git 版本管理工作流。

## 🚀 项目预览
- **在线演示地址**: [点击这里查看我的简历](https://koka-a.github.io/my-resume-web/)

## ✨ 核心功能
- **响应式设计**: 完美适配手机、平板及 PC 端浏览。
- **深色模式切换**: 使用原生 JavaScript 实现的一键换肤功能。
- **语义化 HTML**: 遵循 HTML5 标准，有利于 SEO 和辅助功能。

## 🛠️ 技术栈
- **HTML5**: 页面结构搭建。
- **CSS3**: 负责页面排版、动画及响应式适配。
- **JavaScript**: 实现交互逻辑。
- **Git**: 版本控制与代码托管。

## 📝 开发记录 (Git Workflow)
本项目严格遵循标准开发流程：
1. `git init` 初始化项目。
2. 使用 `git add` 和 `git commit` 记录功能迭代。
3. 通过 `git push` 部署至远程仓库及 GitHub Pages。
